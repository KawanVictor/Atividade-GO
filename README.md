## Atividade Go - Hello World

Este é um projeto simples em **Go (Golang)** criado como uma introdução à linguagem.  
O programa exibe a mensagem **"Hello, world!"** no terminal.

---

##  Como executar

### 1. Pré-requisitos
- [Go](https://go.dev/dl/) instalado na sua máquina (versão 1.18 ou superior)
- Um editor de código, como [Visual Studio Code](https://code.visualstudio.com/)

---

### 2. Clonar o repositório
```bash
git clone https://github.com/seu-usuario/Atividade-GO.git
cd Atividade-GO
